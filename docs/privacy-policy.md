# Політика конфіденційності — First Step

**Дата набрання чинності:** 2026-02-27

## 1. Загальні положення

Додаток **First Step — англійські слова** («Додаток») призначений для вивчення англійської лексики. Ця Політика конфіденційності пояснює, як ми обробляємо інформацію під час використання Додатку.

## 2. Дані, які зберігаються на вашому пристрої

Додаток зберігає такі дані виключно **локально на вашому пристрої** (AsyncStorage):

| Дані                                                               | Мета                                |
| ------------------------------------------------------------------ | ----------------------------------- |
| Прогрес навчання (слова пройдено, правильні/неправильні відповіді) | Відображення статистики             |
| Щоденна ціль та стрік                                              | Мотивація та відстеження активності |
| Налаштування (тема, категорія слів, автоперехід, звук)             | Персоналізація досвіду              |
| Дата першого запуску (прапорець онбордингу)                        | Відображення вітального екрану      |

## 3. Дані, які ми НЕ збираємо

- Ми **не збираємо** жодних персональних даних (ім'я, email, телефон тощо).
- Ми **не передаємо** жодні дані третім особам.
- Ми **не використовуємо** аналітику, рекламні мережі або трекери.
- Ми **не маємо** власних серверів або баз даних.

## 4. Дозволи Android

Додаток використовує лише:

- **VIBRATE** — для тактильного зворотного зв'язку при відповіді на питання.

Жодних інших дозволів (камера, мікрофон, геолокація, контакти тощо) не запитується.

## 5. Зберігання та безпека

Усі дані зберігаються локально на вашому пристрої. Ви можете видалити їх у будь-який момент через:

- Меню → Налаштування → Скинути прогрес (всередині Додатку)
- Або видаливши Додаток з пристрою.

## 6. Діти

Додаток підходить для всіх вікових груп. Ми не збираємо жодних даних від дітей або будь-яких інших користувачів.

## 7. Зміни до цієї Політики

У разі суттєвих змін ми оновимо дату вгорі цієї сторінки. Продовження використання Додатку після оновлення означає прийняття нової версії Політики.
